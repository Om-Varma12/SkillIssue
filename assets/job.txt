Job Title: AI & Machine Learning Engineer (Full-Stack AI Developer)

Location: Pune / Remote
Experience Level: Entry to Mid-level (0–2 years of hands-on project experience)

About the Role

We’re seeking a passionate AI & ML Engineer with strong foundations in machine learning, data-driven application development, and end-to-end AI system deployment. The ideal candidate is a self-driven technologist who has independently built and deployed AI-powered applications integrating both backend intelligence and modern web interfaces.

You’ll work on projects ranging from phishing detection systems and intelligent chatbots to campaign analytics dashboards, helping bring ML research into production environments.

Key Responsibilities

Design, build, and deploy machine learning models for real-world use cases (classification, prediction, anomaly detection, etc.).

Develop AI-powered full-stack applications using Flask (backend) and React (TypeScript) (frontend).

Implement data preprocessing, feature engineering, and model optimization using Python, Scikit-learn, and TensorFlow.

Build explainable AI components for transparency in model decisions.

Create RESTful APIs and integrate ML models into web or browser-based systems.

Collaborate on dashboard and visualization tools for KPI tracking and decision support.

Maintain strong documentation, version control (Git), and deployment pipelines.

Required Skills & Technologies

Languages: Python, JavaScript (TypeScript)

Frameworks: Flask, React

ML Tools & Libraries: Scikit-learn, TensorFlow, Pandas, NumPy

Databases: MySQL

Others: Git, HTML, CSS, REST APIs, Data Visualization (Matplotlib)

Preferred Qualifications

Bachelor’s degree in Artificial Intelligence, Data Science, or Computer Science.

Demonstrated portfolio of AI/ML and full-stack projects (e.g., phishing detection, chatbots, analytics dashboards).

Familiarity with LLMs, Transformers, and Explainable AI concepts.

Experience in end-to-end model deployment and integration into live applications.

Strong analytical, problem-solving, and project execution skills.
